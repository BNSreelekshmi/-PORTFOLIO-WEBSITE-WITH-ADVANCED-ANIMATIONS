*Company Name* :Codtech IT Solutions Private Limited

*Name* : Sreelekshmi B

*Intern ID* :CTIS6069

*Domain* : Web Development

*Duration* : 4 weeks

*Mentor Name* : Neela Santhosh

## The task is to develop a professional portfolio website from scratch that showcases my skills, education, projects, and contact information. The goal is to create a visually appealing and interactive website that represents my abilities as a web developer while demonstrating proficiency in modern front-end technologies. The website includes key sections such as Hero, About Me, Skills, Education, and Projects, all built using clean, semantic code and designed to provide a smooth, engaging user experience.

Tools Used:
HTML – for structuring the website, creating semantic sections, headings, paragraphs, buttons, and cards. It forms the foundation of the portfolio.
CSS – for styling the website with modern design principles, including gradients, spacing, rounded cards, hover effects, and responsive layouts. CSS transitions and animations are used to provide smooth interactions.
JavaScript – for adding dynamic interactivity, handling events like button clicks, and controlling animations.
GSAP (GreenSock Animation Platform) – to create advanced animations, such as fading in text, sliding images, staggered card animations, and scroll-triggered effects, giving the portfolio a professional feel.
Visual Studio Code – as the development environment to write, organize, and test the website efficiently.
Live Server (VS Code Extension) – for real-time preview of changes in the browser.
Why It’s Applicable:
Developing a portfolio from scratch provides hands-on experience in front-end web development and demonstrates practical skills that are highly valued in the industry. The project allows me to combine coding with design principles, producing a website that is functional, visually appealing, and interactive. Using GSAP and CSS animations helps create smooth transitions and modern effects, which are standard in professional web development today.
This task also demonstrates understanding of responsive design and user experience (UX) by ensuring that all sections, images, and content adapt well to different screen sizes. It trains me to structure projects efficiently using proper folder organization and semantic HTML. The ability to develop a polished portfolio from scratch is directly relevant to real-world scenarios, such as building client websites, landing pages, or web applications.
In conclusion, Task 1 focuses on developing a fully functional, modern portfolio website from scratch using HTML, CSS, JavaScript, and GSAP. It highlights practical skills in coding, UI/UX design, and animations while producing a professional project that can be presented to potential employers or clients. This task lays the foundation for advanced web development projects and showcases readiness for front-end development roles.
